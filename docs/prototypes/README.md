# Prototypes

## Naming rules

``` bash
base:${PLATFORM_NAME}-${IMAGE_BASE_NAME}-${IMAGE_BASE_VERSION}
```
* example: base:linux-ubuntu-20.04
* example: base:windows-nanoserver-2004
